## Cydonile 榅枋茶馆 

榅，读音 wēn 和 yùn 均可。读作 wēn 时包含“根”的含义，也可指榅桲树的果实，黄色，有香气，味甘酸，可以吃，又供药用；读作 yùn 时其本义为草木繁盛。

枋，此处读音 fāng。可指筑堤坝等水利工程的大木桩，亦指古书上记载可做车的木材。

Cydonile，源自 Cydonia （榅桲，蔷薇科榅桲属灌木）和 pile （堆，大笔钱财，电池组，动词可指堆叠累计，用桩加固支撑）。

## 与 NoobArchive 的联系

作为于 2021 年开办至今的原 **猪 / 瘟 / 小 / 窝** 站点，[NoobArchive WordPress](https://zw.mb233.net) 曾经在发布组织内容和作品展列方面做了点缓冲，起码能用。

很遗憾，由于实际情况，该站点不适合直接用作杼榅的个人博客站点。

由于一时的配置问题，zw.mb233.net 饱经风霜，频出问题。

我们决定也借此尝试更多的建站方案。

## 好吧，这是为什么

我希望有一个在保持现代安全快速前提下还能有一个简洁清晰的博客站点分享我的笔记或者碎碎念之类的，此外能够得到更多掌控权。

好像真的只需要 Markdown 就能写出我想要的一切，顶多加几个小扩展。

现有的 WordPress 站点实在字面意义的太肥了，Theme Sakurairo 虽然漂亮但是太令人眼花缭乱。当然 ACG 风格都或多或少有这个缺陷。

但我希望保留一个站点用作实验。

认识我的朋友都应该清楚我不希望彻底抛弃某个平台或者项目，至少不会彻底雪藏或删掉，除非有大问题。 甚至 DreamX Network 这个名字最初
就是 2017 年底自建 Minecraft 服务器的又一个“新名字”的重新利用，但之后我不想再随便再更改名称了。


NoobArchive WordPress 站点还会留着，当然 MingbaiCloud 的 [mb233.net](https://mb233.net) 域名也快过期了，不清楚 GZY_mingbai 之后会怎么处理。

无论如何，我很感激 GZY_mingbai 所提供的一切。
